# Estratégia de Multi-idioma (i18n)

## Decisão

- **Idioma padrão do Tchesco OS:** Português (Brasil) — `pt_BR.UTF-8`
- **Idiomas incluídos:** Todos os disponíveis no Ubuntu (~50+ idiomas)
- **ISO estimada:** 4 a 4.5 GB

## Por que incluir todos?

- Usuário escolhe na primeira inicialização sem precisar baixar pacotes
- Pros amigos em diferentes regiões, funciona out-of-the-box
- ISO maior é aceitável (não é restrição pro uso pessoal)
- Menos dor de cabeça pós-instalação

## Camadas de idioma no Linux

Idioma no Linux não é "uma coisa só". São cinco camadas que precisam alinhar:

### 1. Locale do sistema

Define formato de data, número, moeda, ordenação. Configurado via:
- `/etc/default/locale`
- `/etc/locale.gen`
- Comando `locale-gen`

### 2. Tradução da interface

Cada aplicativo pode ser traduzido. KDE usa arquivos `.mo` em `/usr/share/locale/`.

### 3. Input method (para idiomas não latinos)

Japonês, chinês, coreano, árabe, tailandês precisam de software adicional para digitar. Usaremos **fcitx5** como framework padrão.

### 4. Layout de teclado

Cada idioma pode ter layout diferente. Configurado via `systemd-localed` e KDE Settings.

### 5. Fontes

Sem fonte que cobre o alfabeto, o texto aparece como quadradinhos (□□□). Usaremos **Google Noto** que cobre praticamente todos os sistemas de escrita do mundo.

## Implementação no script

### Pacotes que vão no `02b-i18n.sh`:

```bash
#!/bin/bash
# Módulo de internacionalização do Tchesco OS

echo "[Tchesco] Instalando suporte multi-idioma completo..."

# Atualizar lista de pacotes
sudo apt update

# Language packs completos (todos os idiomas)
sudo apt install -y $(apt-cache search language-pack- | grep -v "\-base\|\-gnome" | awk '{print $1}')

# Language packs do KDE Plasma
sudo apt install -y $(apt-cache search plasma-l10n- | awk '{print $1}')

# Traduções de aplicativos essenciais
sudo apt install -y \
    libreoffice-l10n-pt-br \
    libreoffice-l10n-pt \
    libreoffice-help-pt-br \
    libreoffice-help-pt

# Firefox e Thunderbird em vários idiomas
sudo apt install -y \
    firefox-locale-pt \
    firefox-locale-en \
    firefox-locale-es \
    thunderbird-locale-pt-br \
    thunderbird-locale-en-us \
    thunderbird-locale-es

# Fontes que cobrem todos os alfabetos
sudo apt install -y \
    fonts-noto \
    fonts-noto-cjk \
    fonts-noto-cjk-extra \
    fonts-noto-color-emoji \
    fonts-noto-extra \
    fonts-noto-ui-core \
    fonts-noto-ui-extra \
    fonts-noto-unhinted

# Corretores ortográficos
sudo apt install -y \
    hunspell-pt-br \
    hunspell-pt-pt \
    hunspell-en-us \
    hunspell-en-gb \
    hunspell-es \
    hunspell-fr \
    hunspell-de-de \
    hunspell-it

# Input methods para idiomas asiáticos
sudo apt install -y \
    fcitx5 \
    fcitx5-configtool \
    fcitx5-frontend-gtk3 \
    fcitx5-frontend-gtk4 \
    fcitx5-frontend-qt5 \
    fcitx5-mozc \
    fcitx5-chinese-addons \
    fcitx5-hangul \
    fcitx5-unikey

# Definir pt_BR.UTF-8 como padrão
sudo locale-gen pt_BR.UTF-8
sudo update-locale LANG=pt_BR.UTF-8 LC_ALL=pt_BR.UTF-8

# Configurar fcitx5 como input method padrão
im-config -n fcitx5

echo "[Tchesco] Multi-idioma configurado!"
```

## Experiência do usuário

### Primeira inicialização

O Tchesco OS inicia em Português (Brasil). O usuário pode trocar em:

**Configurações do Sistema → Regional Settings → Idioma**

Opções disponíveis: todos os idiomas instalados (lista ~50).

### Trocando de idioma

1. Usuário vai em Configurações → Regional Settings
2. Seleciona idioma preferido
3. Faz logout e login de novo (ou reinicia)
4. Interface inteira traduzida

### Digitando em outro idioma

Para digitar em japonês, chinês, coreano:

1. Configurações → Regional Settings → Input Method
2. Adicionar layout do idioma
3. Atalho: `Ctrl+Espaço` alterna entre layouts
4. fcitx5 mostra indicador na barra do sistema

### Layout de teclado

Usuário pode ter múltiplos layouts simultâneos:
- ABNT2 (Brasil) — padrão
- US International
- Qualquer outro via Configurações

## Tamanho de cada idioma

Cada `language-pack-*` pesa em média 30-80 MB. Somando 50 idiomas + help packs + LibreOffice + Firefox + Thunderbird traduzidos, chega a aproximadamente 1.5 GB. É o que infla a ISO, mas é o preço da universalidade.

Se no futuro a ISO ficar grande demais, consideramos v2 com instalador que baixa só o idioma escolhido.
